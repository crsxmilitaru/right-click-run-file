## âœ¨ Features

- **Context Menu Integration**: Run scripts and executables directly from the explorer.
- **Cross-Platform**: Support for Windows, macOS, and Linux.
- **Run as Admin**: Execute with elevated privileges (RunAs/sudo).
- **Integrated Terminal**: Seamlessly run and view output in VS Code.

## ğŸ“– Usage

1. **Right-click** on any supported file in the VS Code Explorer.
2. Select **Run File** to execute normally.
3. Select **Run File as Admin** to execute with elevated privileges.
